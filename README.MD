# Vagas de Emprego

> Backend desenvolvido em Java + Spring Boot

## O que aprendi neste projeto?

```
Login com Token JWT
Criação de filtros no Spring Security
Relacionamento e mapeamento de Entidades
Queries de consulta ao banco de dados customizadas
Validation
```
